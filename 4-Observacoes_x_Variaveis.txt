
Observações x Variáveis



-> Observação - é uma ocorrência de um item de dados específico que é gravada sobre uma unidade de dados. Também chamada de
                registro. (linhas de uma planilha de excel são as observações)


-> Variáveis  - característica de intereresse que é medida em cada elemento da amostra ou população. Como o nome sugere, seus
                valores variam de elemento para elemento. As variáveis podem ter valores numéricos ou não numéricos.
                (colunas de uma planilha de excel)



- Dependendo do conjunto de dados as observações são as colunas e variáveis são as linhas. 

- Por isso nosso trabalho como Cientista de Dados é garantir que tenhamos observações e variáveis de acordo com o que os 
  dados representam.


- Diante de um dataset, primeiro pergunta seria "estamos trabalhando com população ou amostra". 
- Na sequência "estes dados vieram de fonte primária ou secundária?"
- Por fim identificamos o contexto dos dados e assim identificamos o que é Observação e o que é Variável





***   Tipos de Vairáveis    ***

Qualitativas - utilizam termos descritivos para descrever algo de interesse. 
               Ex: cor dos olhos, estado civil, religião, sexo, grau de escolaridade, classe social, tipo sanguíneo...


Quantitativa - representadas por valores numéricos que pode ser contados ou medidos.
               Ex: número de crianças em uma sala de aula, peso, idade, número de filhos...



Exemplo na prática:

                 Idade		Sexo		Peso		Cor dos Olhos

Indivíduo 2       34             M		 74		  Castanho	
Indivíduo 3       56             F               89                Azul

-> Sexo  e Cor dos Olhos: são variáveis qualitativas
-> Idade e Peso         : são variáveis quantitativas




Dentro das variáveis qualitativas e quantitativas ainda temos uma subdivisão para cada uma:


           Qualitativas                                    Quantitativas

    Nominais         Ordinais                         Discretas        Contínuas                                                                   
    Profissão      Escolaridade			     Nº de filhos       Altura
    Sexo           Classe Social		     Nº de carros	Peso

-> Um dado classificado como "idade" é quantitativo
   Ex: 11, 15, 25, 42 anos.

-> Se o dado "idade" for informado por "faixa etária", então ele é qualitativo (ordinal), pois neste caso temos uma    hierarquia.
   Ex: 0 - 5 anos. 6 - 12 anos. 13 - 18 anos...
  




Vamos explorar um pouco mais sobre esses tipos de variáveis:

Qualitativas: As variáveis qualitativas são características que não podem ser quantificadas numericamente, mas são descritas por meio de termos ou categorias. Podem ser divididas em duas subcategorias:

Qualitativas nominais: Nesse caso, as categorias não possuem uma ordem específica. Exemplos são o sexo, cor dos olhos, estado civil, religião, etc. Os dados são classificados em grupos distintos, mas não há uma ordem hierárquica entre eles.

Qualitativas ordinais: Nesse caso, as categorias possuem uma ordem hierárquica específica. Exemplos incluem grau de escolaridade (primário, secundário, superior) ou classes sociais (baixa, média, alta). As categorias podem ser ordenadas de acordo com algum critério, mas a diferença entre elas não é necessariamente quantificável.

Quantitativas: As variáveis quantitativas são medidas numéricas que podem ser contadas ou medidas de forma precisa. Podem ser divididas em duas subcategorias:

Quantitativas discretas: São variáveis que representam contagens inteiras ou números inteiros específicos. Exemplos incluem o número de crianças em uma sala de aula, o número de filhos de uma família, etc.

Quantitativas contínuas: São variáveis que podem assumir valores em uma escala contínua. Elas são medidas e podem assumir qualquer valor dentro de um intervalo. Exemplos incluem peso, altura, idade, temperatura, etc.
